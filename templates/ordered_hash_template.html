Duplicate Email Removal Simulation

This program sorts 100k random email user ids, with duplicates. It generates 12 character random email IDs & repeats them randomly with duplicates.

To remove duplicates, I have implemented custom hashmap - OrderedHash

  • Iterate over email list
  • Implements insert, hash & rehashing to only store unique values
  • When inserting a unique value it will update (in order) an internal array
  • In order array does not get updated if a duplicate id insertion is attempted
